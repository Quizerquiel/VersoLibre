# VersoLibre

Blog de poemas hecho con Svelte y Vite para que las personas publiquen sus versos desde el navegador.

## Incluye

- interfaz editorial construida en un componente Svelte
- formulario para publicar poemas
- guardado local con `localStorage`
- buscador por titulo, autor, tema y contenido
- filtro por categoria
- poemas de ejemplo para arrancar

## Desarrollo

1. Instala dependencias con `npm install`.
2. Inicia el entorno local con `npm run dev`.

## Build

- `npm run build`
- `npm run preview`

## Nota

Esta version sigue siendo frontend puro. Los poemas se guardan localmente en el navegador hasta que decidamos conectarla a una base de datos real.
